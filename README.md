## 🚀 Featured Project

# 🧠 CodeAtlas

> ⚡ **Your AI-powered developer knowledge engine**

### 🔥 What it does

* 📂 Converts scattered code/docs into **structured knowledge**
* 🔍 Enables **semantic search using embeddings**
* 🧠 Uses **vector storage + cosine similarity**
* ⚡ Helps devs instantly **understand large codebases**

---

### 🛠️ Tech Behind It

* Frontend: React
* Backend: Node.js + Express
* AI: OpenAI API (embeddings + retrieval)
* Database: Vector-based storage

---

### 🎯 Why it's cool

* Solves a **real developer pain point**
* Works like a **personal ChatGPT for your codebase**
* Built with **modern AI architecture (RAG)**
